🚀 TaskFlow Lite

A lightweight and responsive task management web application built using HTML, CSS, and JavaScript.
TaskFlow Lite allows users to efficiently manage daily tasks with persistent storage using the browser’s localStorage.

---

📌 Live Demo

👉 https://payalpr2003-stack.github.io/taskflow-lite/

---

✨ Features

- ➕ Add new tasks
- ❌ Delete tasks
- 💾 Persistent data storage using localStorage
- ⚡ Instant UI updates (DOM manipulation)
- 📱 Simple and responsive interface

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Storage: Browser LocalStorage
- Deployment: GitHub Pages

⚙️ How It Works

1. User enters a task in the input field
2. Task is validated before adding
3. Task is stored in localStorage
4. UI updates dynamically using JavaScript
5. Tasks persist even after page refresh
